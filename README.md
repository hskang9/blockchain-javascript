## Installation

1. Make sure [node.js 7.2+](https://nodejs.org/en/) is installed. 
2. Install [npm](https://www.npmjs.com/) and packages in package.json file. 

```
$ npm install
```

3. Run the server:
```
$ node index.js 
```
